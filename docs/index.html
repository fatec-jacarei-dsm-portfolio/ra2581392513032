Disseminação de Dados Limnológicos

Projeto acadêmico em cooperação com INPE, UFRJ, UFJF e IIE

ABP 2025.2

Projeto acadêmico voltado à disponibilização e análise de dados limnológicos e meteorológicos de reservatórios da Furnas. O sistema integra dados de coletas manuais (campanhas) e medições automáticas em tempo real (SIMA), apoiando estudos sobre balanço de carbono.
Atuei no desenvolvimento do back-end, construindo uma API RESTful para consulta dos dados, e no front-end, implementando funcionalidades e estilização. A aplicação consome três bancos de dados fornecidos pelo INPE, containerizados com Docker, transformando os dados em mapas, gráficos e tabelas interativas.

Tecnologias utilizadas: React, TypeScript, Node.js, Express, PostgreSQL, Docker, Chart.js, Leaflet

Detector de Plágio

Aplicação de análise de similaridade textual com PLN

2025.2

Aplicação web interativa para análise de similaridade textual entre documentos utilizando o Índice de Jaccard. A ferramenta permite refinar os resultados com técnicas de Processamento de Linguagem Natural (PLN).
O usuário pode aplicar filtros como remoção de stopwords, stemming e análise por bigramas, além de processar textos via input ou upload de arquivos (.txt). O sistema realiza o cálculo da similaridade entre os conjuntos gerados e apresenta os resultados de forma clara.

Tecnologias utilizadas: HTML, CSS, JavaScript
