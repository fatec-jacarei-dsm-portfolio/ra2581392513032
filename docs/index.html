Sobre Mim

No segundo semestre da minha jornada acadêmica, venho aprimorando minhas habilidades na criação de experiências digitais completas. Minha trajetória começou com o desenvolvimento frontend, onde aprendi a dar vida a layouts e a criar interfaces que fluem de forma intuitiva para o usuário.

Atualmente, meu foco se expandiu para o backend, onde a complexidade e a arquitetura de dados me atraem particularmente. Tenho um grande interesse em sistemas de gerenciamento e manipulação de informações, vendo a base de dados como a espinha dorsal de qualquer aplicação robusta e eficiente.

Além da minha formação, tive a oportunidade de vivenciar um ambiente corporativo utilizando o sistema SAP para realizar consultas, entradas e atualizações de dados. Essa experiência me proporcionou uma base sólida em atenção aos detalhes e na compreensão da dinâmica de sistemas empresariais complexos.

Disseminação de Dados Limnológicos

Projeto acadêmico em cooperação com INPE, UFRJ, UFJF e IIE

Projeto acadêmico com foco em disponibilizar e analisar dados limnológicos e meteorológicos de reservatórios da Furnas Centrais Elétricas S.A. O objetivo é subsidiar estudos sobre o Balanço de Carbono, disponibilizando os dados de forma organizada, integrando coletas manuais (campanhas) e dados automáticos em tempo real (SIMA).
Participei no desenvolvimento do back-end, construindo a API RESTful para consulta dos dados, e atuei no front-end na implementação de funcionalidades e estilização. O sistema consome 3 bancos de dados (disponibilizados pelo INPE) containerizados com Docker, traduzindo as informações armazenadas para mapas, gráficos e tabelas visuais.

Tecnologias utilizadas: React, TypeScript, Node.js, Express, PostgreSQL, Docker, Chart.js, Leaflet

Detector de Plágio

Uma aplicação web interativa que analisa a similaridade textual entre dois documentos usando o Índice de Jaccard. A ferramenta vai além da matemática básica, permitindo ao usuário aplicar técnicas de Processamento de Linguagem Natural (PLN) para refinar a análise. O usuário pode determinar a resposta através de filtros avançados como: Remover StopWords, Adicionar Stemming e Analisar por Bigrams. Ele processa textos (via input/upload de .txt) e realiza o cálculo da similaridade entre os conjuntos apresentados.

Tecnologias utilizadas: HTML, CSS, JavaScript
